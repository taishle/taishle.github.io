# taishle.github.io

Personal academic website of **Tai Le**, Biswas Postdoctoral Fellow at the MIT Perceptual Engineering Lab / MIT HEALS Collaborative.

Live site: https://taishle.github.io

## Pages

- **Home** — Bio, research interests, and news
- **Publications** — Journal papers, conference papers, book chapters, and preprints
- **Research** — Research projects with descriptions and images
- **Professional Activities** — Reviewer roles, oral presentations, and PI history
- **Contact** — Email, GitHub, LinkedIn, Google Scholar

## Tech

Plain HTML/CSS/JS — no frameworks, no build step. Dark mode toggle stored in `localStorage`. Deployed via GitHub Pages on push to `main`.
